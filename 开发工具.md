
| 工具                 | 本质是什么              | 一句话定位                | 最核心差别                |
| ------------------ | ------------------ | -------------------- | -------------------- |
| **Cursor**         | AI 原生代码编辑器         | “把 AI 直接变成你的 IDE 大脑” | 能理解**整个项目**并直接改多文件   |
| **Trae**           | AI 编程 IDE（国产）      | “中文更友好的 AI 工程助手”     | 更偏**工程化、自动改项目结构**    |
| **Antigravity**    | AI 自动化编程工具         | “让 AI 主动完成大规模代码改动”   | 更强调**任务级 / 重构级自动执行** |
| **VS Code**        | 通用代码编辑器            | “最强底座，但自己不带 AI”      | 只是平台，能力靠插件           |
| **Codex**          | AI 编程模型 / Agent 能力 | “能被用来自动写和跑代码的引擎”     | 偏**底层能力**，不是完整 IDE   |
| **Claude Code**    | AI 编程 Agent（CLI）   | “会读大仓库、能做复杂修改的 AI”   | **长上下文 + 深度理解项目**    |


## VS Code 推荐 AI 插件清单
| 插件名称                          | 用途                  | 适合场景        |
| ----------------------------- | ------------------- | ----------- |
| **GitHub Copilot**            | AI 代码补全、建议、函数自动填充   | 日常编写代码      |
| **Copilot Chat**              | 解释代码、提问式调试、代码片段生成   | 深入理解、QA 调试  |
| **Continue**                  | 让 AI 生成/修改多个文件 diff | AI 驱动的项目级改动 |
| **Claude 3 / Claude Code 插件** | 强上下文理解、解释汇总、复杂改动建议  | 大项目代码理解     |
| **Sourcegraph Cody**          | 全仓库搜索 + 智能理解        | 结构复杂项目理解    |
| **Tabnine AI Autocomplete**   | 补全建议（可与 Copilot 叠加） | 补全增强        |
| **Codeium**                   | 本地/云 AI 辅助补全        | 可免费替代补全     |
| **Error Lens**                | 强调错误/警告在行内显示        | 代码问题可视化     |
| **GitLens**                   | 强化 Git 历史/作者/代码来源   | 管理代码历史      |
