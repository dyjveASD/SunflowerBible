# 我的金融感想
## 投资是什么？
投资 是一个“投资系统工程”，而不是“赚快钱”。
不是：押方向、猜涨跌
而是：有一套输入 → 处理 → 输出 → 复盘的机制
投资底层认知：政策的作用是“降低最坏情况的概率”，而不是“制造最好的情况”。

## 经济周期 × 政策行为 对照表
| 经济状态 | 常见现象      | 政策取向         | 你能观察到的信号  |
| ---- | --------- | ------------ | --------- |
| 衰退   | 需求不足、失业压力 | 降息 / 放水 / 刺激 | 降准、降息、稳增长 |
| 复苏   | 数据改善但不稳   | 偏宽松          | 定向刺激、呵护市场 |
| 过热   | 通胀压力      | 加息 / 收紧      | 强调防风险、控杠杆 |
| 下行   | 增速放缓      | 转向宽松         | 政策态度变“软”  |

看：政策在干什么。 
政策 = 最诚实的经济温度计

## 个人阶段目标（V1.0）
我能通过中国的宏观政策或经济新闻，判断：

1）政府/央行在解决什么问题

2）经济大致处在什么阶段

3）哪些大类资产更容易受益或受损

4）我个人需要“行动 / 不行动 / 防风险”

## 分析模型
【宏观事件】
    ↓
【政府 / 央行在担心什么？】
    ↓
【用了什么政策工具？】
    ↓
【说明经济处在什么阶段？】
    ↓
【影响哪类资产？】
    ↓
【普通人层面的含义】

## 「30 秒判断宏观新闻是否值得看」的方法
1️⃣ 是不是央行 / 国务院 / 财政部？
2️⃣ 有没有动钱？（利率 / 流动性 / 财政）
3️⃣ 是不是在“防风险 / 托底 / 稳预期”？
